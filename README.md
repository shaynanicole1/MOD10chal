# 👨‍💼 Employee Tracker

a command-line application from scratch to manage a company's employee database, using Node.js, Inquirer, and PostgreSQL.

---

## 📦 Tech Stack

- Node.js
- MySQL2
- Inquirer
- console.table

---

## 🚀 Features

- View all departments
- View all roles
- View all employees
- Add a department
- Add a role
- Add an employee
- Update an employee's role
